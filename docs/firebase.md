# Firebase

## メール/パスワードでの認証準備
- (Firebase コンソール)[https://console.firebase.google.com/?hl=ja]でアプリのプロジェクトに移動
- メールアドレスとパスワードによるログインを有効にする
  - Firebase コンソールで [Authentication] セクションを開く
  - [Sign-in method] タブで [メール / パスワード] を有効にして、[保存] をクリック
